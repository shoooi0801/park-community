# 公園community
​
## サイト概要
### サイトテーマ
近所の公園を探したり、自分が行った公園を投稿できるコミュニティサイトです
​
### テーマを選んだ理由
公園が好きで公園巡りなどをすることが多いのですが、どこに公園があるのか、または駐車場などがあるかどうかで悩んでました。
身近に自分と同じように、公園巡りが趣味の友人がいないため困っていた所、他の人がどのような公園に行っているのか、またその公園にはどのような設備があるのか知れるサービルがあれば便利だと考えました。
また公園巡り好きの方だけではなく、育児や保育でどこに公園があるかを知る必要がある人の助けにもなると考えこのテーマにしました。
​
### ターゲットユーザ
 - 公園が好きで、いろいろな公園巡りをしたいと考えてる人
 - 育児や仕事で公園に行く機会がある人
​
### 主な利用シーン
 - 公園巡りをする時
 - その公園に行った人自身の感想を知りたい時
 - 自分が行った公園を投稿する時
